Step 1: Install OpenClaw

You need Node.js version 22 or higher. Check yours with node --version in your terminal.

On Mac or Linux, run:
curl -fsSL https://openclaw.ai/install.sh | bash
On Windows (PowerShell):
1
Choose an AI model** (Claude, GPT, or Gemini). You'll authenticate via OAuth or paste an API key
2
Set up the Gateway** (the background daemon that keeps OpenClaw running)
3
Connect a channel** (start with one, Telegram is easiest)
4
Configure skills** (skip for now, add later)

Setup Step 3

Step 3: Connect Your First Channel

This is where it gets real. Start with ONE channel. The onboarding wizard will ask which one.

Telegram (recommended first channel, simplest setup):
1
Open Telegram, search for @BotFather
2
Send `/newbot` and follow the prompts
3
Copy the bot token
4
Paste it into the OpenClaw wizard
5
Send `/start` to your new bot in Telegram
6
Start chatting
WhatsApp (most useful for business, slightly harder setup):
1
Uses the Baileys bridge (built into OpenClaw)
2
Scan a QR code from your terminal
3
Stores more state on disk than Telegram
4
Takes a few minutes longer but worth it if WhatsApp is your main tool
Or just use the web dashboard (zero channel setup):
openclaw dashboard
🔑

Key Takeaway: Start with Telegram (fastest) or the web dashboard (no setup). Add WhatsApp and others once you're comfortable.

Setup Step 4

Step 4: Set Up Cron Jobs (The Real Power Move)

Skills let OpenClaw do things. Cron jobs let it do things on a schedule, without you asking.

Example: Daily morning briefing

Just message your bot: "Every morning at 6:30am, send me a summary of today's calendar, the weather in Brisbane, and the top 3 AI news stories."

Example: Weekly analytics report

"Every Friday at 5pm, check my YouTube channel stats and email me a summary of views, subscribers, and top performing videos this week."

Other cron job ideas:
Daily lead follow-up reminders
Weekly competitor monitoring summaries
Monthly invoice reminders
Daily social media
engagement checks
