Before You Start

What You Need Before Starting

Three things. That's it. 1. A computer (Mac, Windows, or Linux. Mac is easiest.) 2. Node.js version 22 or higher. Don't know if you have it? Open your terminal and type: ``` node --version ``` If it says `v22` or higher, you're good. If it says "command not found" or a lower number, go to nodejs.org, download version 22, install it, then come back. 3. An AI account. You need one of these (pick one):

OpenAI account (chatgpt.com)
— easiest setup, uses OAuth
Anthropic account (console.anthropic.com)
— best reasoning, needs API key
Google account (aistudio.google.com)
— free tier available, needs API key
How to open your terminal:
Mac: Press Cmd
+ Space, type "Terminal", hit Enter
Windows: Press Windows
key, type "PowerShell", hit Enter
Linux: You already know

Step 1

Install

Step 1: Install OpenClaw (2 minutes)

Open your terminal. Copy and paste this command. Hit Enter.

Mac or Linux:
curl -fsSL https://openclaw.ai/install.sh | bash
Windows PowerShell:
1
The terminal will show you a line starting with `export PATH=`
2
Copy that entire line
3
Open a new terminal window
4
Type `nano ~/.zshrc` (Mac) or `nano ~/.bashrc` (Linux)
5
Paste the line at the bottom
6
Press Ctrl+X, then Y, then Enter to save

Step 2

Onboarding Wizard

Step 2: Run the Setup Wizard (10 minutes)

This is the big one. The wizard asks you a series of questions. I'll tell you exactly what to pick.

Type this and hit Enter:

openclaw onboard --install-daemon
Screen 1: Security consent

You'll see: "I understand this is powerful and inherently risky. Continue?"

Screen 2: Onboarding mode

You'll see two options:

  • QuickStart
  • Advanced
Screen 3: Model / Auth provider

You'll see a list of AI providers:

OpenAI
Anthropic
Google
Custom / Local
Screen 4: Default model

It'll suggest a default model. Select Keep current. Press Enter.

Screen 5: Channel selection

It'll ask which messaging channel to connect. For now, select Skip for now.

Screen 6: Skills and extras

A few more prompts will fly by:

"Install missing skill
dependencies?" → Select Skip for now
"API keys for
optional skills?" → Select No
"Hooks?" → Select
Skip for now
Screen 7: Done!

The wizard will finish and show you a section that says "Dashboard ready" with a link like:

COPY THIS LINK. Save it somewhere. This is your dashboard URL with your access token. You need it for the next step.

🔑

Key Takeaway: QuickStart mode, connect your AI account via OAuth or API key, skip channels and skills for now. Save the dashboard link.

Step 3

Your First Chat

Step 3: Open the Dashboard and Say Hello (1 minute)

Open your browser. Paste in the dashboard link you saved from the wizard. It looks like: ``` http://127.0.0.1:18789/?token=YOUR_TOKEN_HERE ``` A chat interface loads. Type "hello" and press Enter. If OpenClaw responds, congratulations. You have an AI agent running on your machine. Try asking it something useful:

"What day is
it today?"
"Summarise this website:
[paste any URL]"
"Write me a
short email to a client thanking them for their time"

Step 4

Connect Telegram

Step 4: Connect a Messaging Channel (5 minutes)

Now let's connect it to a messaging app so you can chat with your AI agent from your phone. Telegram is the easiest first channel.

4a. Create a Telegram bot:
1
Open Telegram on your phone (download it free if you don't have it)
2
Search for @BotFather (it has a blue checkmark)
3
Tap Start
4
Type `/newbot` and send it
5
BotFather will ask "What name do you want for your bot?" Type any name, e.g. `My AI Assistant`
6
BotFather will ask for a username. It must end in "bot". Type something like `myai_openclaw_bot`
4b. Connect it to OpenClaw:

Go back to your terminal. Type:

openclaw onboard

When it asks about channels, select Telegram. Paste your bot token when prompted. Press Enter.

4c. Test it:
1
Open Telegram
2
Search for your bot by the username you created (e.g. `myai_openclaw_bot`)
3
Tap Start
4
Type "hello"
5
Your AI agent should reply

Step 6

Keep It Running 24/7

Step 6: Make It Run All The Time (Optional But Recommended)

Right now, OpenClaw only works when your computer is on and the terminal is open. Here's how to fix that.

Option A: Mac Mini (easiest, no technical skill needed)
1
Get a Mac Mini (or use a spare one). Even an old one works
2
Plug it in and connect to wifi
3
Download "Amphetamine" from the App Store (free). This stops the Mac from sleeping
4
Open Amphetamine, right-click the icon in the menu bar, select "Start New Session" > "Indefinitely"
5
Open Terminal, run the install and onboard commands from Steps 1-2
6
Close the lid. It keeps running. Done
Option B: VPS / Cloud Server ($5-20/month)
1
Sign up at digitalocean.com (or hetzner.com for cheaper)
2
Create a "Droplet" with Ubuntu. Pick the $10/month option (2 CPU, 2GB RAM works fine)
3
SSH into your server: `ssh root@YOUR_SERVER_IP`
4
Install Node 22: `curl -fsSL https://deb.nodesource.com/setup_22.x | bash - && apt-get install -y nodejs`
5
Install OpenClaw: `curl -fsSL https://openclaw.ai/install.sh | bash`
6
Run: `openclaw onboard --install-daemon`

Security

Security: Read This Before You Get Too Excited

OpenClaw can run commands on your machine. That's what makes it powerful. It's also what makes it risky if you're careless.

Rule 1: Use a dedicated machine

Don't run it on the same laptop where you have your banking passwords saved. Use a Mac Mini, a VPS, or at least a separate user account on your computer.

Rule 2: Give it its own accounts

Create a fresh Gmail for your bot. A fresh Apple ID if it needs one. Don't hand it your personal logins.

Rule 3: Update it

Earlier versions had a security bug (CVE-2026-25253) where a malicious website could hijack your session. Fixed in version 2026.1.29.

Rule 4: Don't put it on the public internet

The Gateway runs on localhost (127.0.0.1) by default. Leave it that way. If you need remote access, use an SSH tunnel (shown in Step 6), not by exposing the port.

Rule 5: DM pairing is your friend

By default, unknown people who message your bot get a pairing code. Their messages don't get processed until you approve them.

🔑

Key Takeaway: Dedicated machine, fresh credentials, keep updated, don't expose to internet. Follow these four rules and you're fine.
